# ðŸ›  **HOW TO CHANGE FILE NAMES IN THE PROGRAM (SETTINGS MENU)**

Inside the program:

```
4. Settings
```

Then you will see these options:

### **1. Change National Anthem file**

Enter new filename or full path.
Example: `C:\Songs\NewAnthem.mp3`

### **2. Change Assembly Bell file**

Enter new filename.
Example: `mybell.mp3`

### **3. Set Extra Audio 1**

Enter ANY file name.
Example: `motivational_speech.mp3`
This becomes **Extra Audio 1** in Assembly menu.

### **4. Set Extra Audio 2**

Enter ANY file name.
Example: `special_event_song.mp3`
This becomes **Extra Audio 2** in Assembly menu.

### **5. Change Prayer/Birthday files for a specific day**

You will choose:

```
0 = Monday
1 = Tuesday
2 = Wednesday
3 = Thursday
4 = Friday
```

Then you can change:

* Prayer file
* Birthday file
* Label (English Day / Hindi Day / Malayalam Day)
